创建一个新的工程¶

RiskQuantLib以工程的方式来进行数据分析。对于任何的数据分析任务，你应该首先创建一个工程。（至少一个工程）

你可以通过终端命令来新建RiskQuantLib工程，新建工程的命令为 newRQL，它的后面应该紧接一个文件夹地址，你可以像这样使用：:

newRQL your_project_path

注意：如果你把一个已经是RiskQuantLib工程的文件夹地址传给 ``newRQL`` 命令，所有的旧文件都会被删除，并且用新的工程文件替换。

当运行这条命令之后，你的文件夹应该多出许多文件，文件夹现在看起来像这样：:

--your_project_path
  --RiskQuantLib
  --build.py
  --main.py
  --Build_Attr.xlsx
  --Build_Instrument.xlsx

在这之后你可以编译这个工程，来使得它符合你的数据处理任务的要求，我们将在下一章解释如何编译工程。
